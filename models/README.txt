
The trained model files are not included in this repository due to GitHub storage limitations.


Download the model package from:

https://drive.google.com/file/d/1MuU_cH10XuwaE0jTnBjqmLoBEOvkValW/view?usp=sharing


After downloading and extracting the package, place the following files inside the `models/` directory:

* TSM_Violence_Model_Final.pth
* yolov8s-pose.pt

## Expected Directory Structure

models/
├── TSM_Violence_Model_Final.pth
└── yolov8s-pose.pt

The application will not run unless both model files are present in the `models/` folder.
